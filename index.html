Hello, Godot! Game Jam 第二屆 即將登場！

想要試試自己做遊戲？想在一個月內完成作品、結交志同道合的夥伴？這個活動就是為你準備的！

Hello, Godot! 是一場以 Godot 引擎 為主軸、線上進行 的 Game Jam 活動。不論你是新手或老手、個人或團隊，都歡迎加入挑戰！

活動時程

  • 7/4（五）20:30 - 主題公布
  • 7/12（六）14:00 - 概念發表
  • 7/19（六）14:00 - DEMO 發表
  • 8/3（日）14:00 - 成果發表

這一屆的新特色

  • 活動延長為整整 一個月，給你更充足的時間打磨作品
  • 定期進度提醒，營造實體 Jam 的緊湊感
  • 搭配 成就牆，完成特定目標能展示你達成的挑戰
  • 報名可選擇是否想組隊、是否想和同地區的人分組
  • 改用 喜愛票選 取代評分制度，鼓勵彼此支持與交流
