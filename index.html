Hello, Godot! Game Jam 第二屆即將開跑！

想體驗親手做出一款遊戲的過程嗎？想在一個月內完成專案，並找到志同道合的夥伴一起交流與成長？
無論你是剛入門的初學者，還是經驗豐富的開發者，只要對 Godot 有興趣，都歡迎加入挑戰。

什麼是 Hello, Godot! Game Jam？

這是一場限定使用 Godot 引擎開發的線上遊戲創作挑戰活動。你可以選擇單人參賽，也可以組隊完成。在這段活動期間內，參與者以指定主題為基礎，設計並完成一款小型遊戲。

Game Jam 不只是限時開發挑戰，更是一個讓你實踐創意、分享成果、認識夥伴的平台。你可以透過本活動嘗試不同的開發流程、測試新的想法，甚至跨出舒適圈挑戰未曾涉略的技術。完成後，作品可以上傳至 itch.io 開放給大家試玩。

此外，在 Discord 群組也設有專區供大家分享開發歷程、技術問題與心得感想，讓開發不再是孤軍奮戰，而是一場有交流、有支持的旅程。
